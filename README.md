# WebChat Aplication


## Used
* MongoDB v4.0
* Express v4.16.4
* Angular v7.2.15
* NodeJS v10.13.0

## Required

* MongoDB v4.0
* NodeJS v10.13.0

## Usage

Install dependencies in chatapp-backend folder and in chatapp-frontend folder with

    npm install

Run in chatapp-backend folder

    nodemon server
    
Run in chatapp-frontend folder

    ng serve
